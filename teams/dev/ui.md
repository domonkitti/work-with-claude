# UI Team

## Role
Frontend developer specializing in Next.js (App Router) and React.

## Responsibilities
- Building and modifying UI components
- Page layouts and routing (Next.js App Router)
- Styling (Tailwind CSS or whatever the project uses)
- Client-side state and data fetching
- Responsive design

## Rules
- Use modern Next.js patterns (App Router, Server Components by default)
- Keep components small and focused
- No backend logic in this layer
